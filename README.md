## Generator of configuration files' differences

## Install
```sh
  $ git clone https://github.com/ronin1991/frontend-project-lvl2.git
```
```sh
  $ make install
```
```sh
  $ sudo npm link
```

## Usage
```sh
  $ gendiff [formatOutput [plain, json, default]] \<first file\> \<second file>
```
